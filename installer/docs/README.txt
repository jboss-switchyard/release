SwitchYard JBoss AS7 Standalone Installer
=========================================

This package contains resources for installing SwitchYard into a JBoss AS7 installation.

Prerequisites:
    1.  Java Runtime
    2.  Fully installed Ant available on the command line execution PATH.
    3.  A JBoss AS7 installation.

Instructions:
    1.  Open a terminal command prompt.
    2.  Change directory into the root of the bundle.
    3.  Execute command "ant".

This script will ask you for the path to the JBoss AS7 distribution and will install all the necessary files
required to run SwitchYard applications on JBoss AS7.  It will also unzip the quickstart examples and forge
tooling resources into the directory containing this README file.
