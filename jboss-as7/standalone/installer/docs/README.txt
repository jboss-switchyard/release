SwitchYard JBoss AS7 Standalone Installer
=========================================

This package contains resources for installing SwitchYard into a JBoss AS7 installation.

Prerequisits:
    1.  Java Runtime
    2.  Fully installed Ant available on the command line execution PATH.

Instructions:
    1.  Open a termal command prompt.
    2.  Change directory into the root of the bundle.
    3.  Execute command "ant".
